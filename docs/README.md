## screen-definitions/ — 화면 정의서

`00_화면목록.md`가 전체 목록, `_규칙.md`·`_화면정의서_템플릿_v6.md`가 작성 기준이다.
정의서를 고친 뒤에는 반드시 `python3 _build-data.py`로 `js/screen-defs-data.js`를 재생성한다(뷰어가 이 파일을 읽는다).

> **알려진 미갱신** — 안전보건교육 정의서는 아직 구 "단일 화면 3탭" 기준이다. 현행 7메뉴 구조에 맞춰 원본을 고친 뒤 재생성해야 한다.
