# 담양군 프로토타입 v2 — 디자인 토큰 기준 (Canonical)

> 이 문서는 프로토타입 v2가 따르는 **디자인 토큰 단일 출처**다. 최종 권위는 `damyang-safety/DESIGN.md` + 실제 토큰(`globals.css`)이며, 본 문서는 그것을 정적 HTML 프로토타입에 이식한 기준이다.
> **규칙: 임의 px·굵기·색상 하드코딩 금지.** 아래 스케일/토큰만 사용한다. 값이 애매하면 최근접 캐논값으로 스냅한다.

작성일: 2026-06-30 · 기준: `css/style.css :root`

---

## 1. 폰트

- **Pretendard Variable** (fallback: Pretendard → -apple-system → system-ui → Apple SD Gothic Neo → Noto Sans KR)

## 2. 타입 스케일 (px) — CSS 변수 `--fs-*`

| 토큰 | px | 역할 |
|---|---|---|
| `--fs-12` | 12 | 캡션·헬퍼·메타·마이크로 라벨 (최소 크기, 이보다 작게 금지) |
| `--fs-13` | 13 | 보조 본문·표 셀·칩 |
| `--fs-14` | 14 | 본문·표 헤더·라벨 (DESIGN.md 핵심 토큰) |
| `--fs-15` | 15 | 본문 강조 |
| `--fs-16` | 16 | 섹션 헤더·카드 타이틀 (DESIGN.md 핵심 토큰) |
| `--fs-18` | 18 | 소제목 |
| `--fs-22` | 22 | 카드 수치·중제목 |
| `--fs-26` | 26 | 대시보드 대형 수치 |
| `--fs-32` | 32 | 강조 수치·서브 타이틀 |
| `--fs-38` | 38 | 페이지 메인 타이틀 (DESIGN.md 핵심 토큰, 최대) |

> ❌ 사용 금지(이탈값): 9 / 9.5 / 10 / 10.5 / 11 / 11.5 / 12.5 / 13.5 / 14.5 / 17 / 20 / 21 / 24 / 28 / 30 / 40 등. 최근접 캐논값으로 스냅(예: 11→12, 12.5→12, 13.5→13, 17→18, 20→22, 24→26, 40→38).

## 3. 폰트 굵기 — CSS 변수 `--fw-*`

| 토큰 | weight | 역할 |
|---|---|---|
| `--fw-regular` | 400 | 본문 |
| `--fw-medium` | 500 | 라벨·필터·보조 강조 |
| `--fw-semibold` | 600 | 섹션 헤더·카드 타이틀·버튼 |
| `--fw-bold` | 700 | 페이지 타이틀·핵심 수치 강조 |

> ❌ 사용 금지: 800/900(→700), `normal`(→400), `bold`(→700). 강약은 4단계(400·500·600·700)로만 표현.

## 5. UX 라이팅 (담양군 공공 톤)

- **간결한 존댓말 안내**: "~하세요", "~되었습니다", "~할 수 있습니다". 명령형 단독("입력") 대신 "제목을 입력하세요".
- **버튼은 동사형**: `등록` `저장` `처리` `점검결과지 생성` `완료 처리`. 모호한 `확인` 남발 지양(맥락 있는 동사 우선).
- **상태·결과 일관 용어**: 접수대기 / 점검중 / 개선조치중 / 완료 / 반려 · 적합 / 미흡 / 해당없음. 동일 개념에 다른 단어 혼용 금지.
- **오류 메시지 = 원인 + 해결**: "제목을 입력하세요"처럼 무엇을 어떻게 할지 명시.
- **강약은 굵기·크기로**: 색상만으로 의미 전달 금지(아이콘/텍스트 병행). 타이틀 700/24~38, 본문 400/14~16, 라벨 500/12~14.

## 6. 적용 방법

```css
/* 예: 카드 타이틀 */
.card-title { font-size: var(--fs-16); font-weight: var(--fw-semibold); color: var(--main-dark2); }
/* 예: 캡션 */
.hint { font-size: var(--fs-12); font-weight: var(--fw-regular); color: var(--text-gray); }
```

신규 화면·컴포넌트는 이 토큰만 사용한다. 리뷰 시 이탈값(비캐논 px/weight, 원시 브랜드 hex) 발견하면 스냅·토큰화한다.
