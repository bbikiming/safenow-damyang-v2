- 첨부파일은 브라우저에서 실제 선택·검증하지만 서버 업로드 없이 파일명만 로컬에 보관한다.
- 현재 브라우저 자동화 환경은 로컬 파일 주입을 지원하지 않아 `증빙 없음` 차단은 실행 검증했고 확장자·20MB·10개 제한은 코드 경로로 확인했다.
- 문자 발송과 온나라 결재는 외부 시스템 연계 전 시뮬레이션이며, 실제 발송·결재를 수행하지 않는다.
- 개인별 관리 기준 시간은 시연용 설정값이며 실제 적용 기준은 발주처 협의가 필요하다.
- 현재 프로토타입은 이름+부서와 누적시간으로 이수현황을 계산한다. 구축 단계에서는 고유 인사 ID와 교육별 이력 원장을 사용해야 인사이동·동명이인·정정 이력을 안전하게 처리할 수 있다.
- 모바일은 반응형 CSS를 적용했으나 이번 현재 실행 검증은 1280×720 브라우저에서 수행했다.
