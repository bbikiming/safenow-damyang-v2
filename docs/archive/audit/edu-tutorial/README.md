# 안전보건교육 관리 UX 검증 기록

- 검증일: 2026-07-16
- 범위: 안전보건교육 7개 하위 메뉴의 핵심 업무 흐름과 초보자용 튜토리얼
- 사용자 목표: 법령 비전문가가 대상자 누락 없이 교육을 계획·실시·증빙하고 미이수자를 후속조치하기

## 검증한 흐름

1. **전체 현황 확인 — 양호**  
   부서별 완료율에서 조치 우선순위를 찾고 대상자별 상세로 이동할 수 있다.
2. **근로자 명단 정비 — 양호**  
   인사연동 읽기 전용 대상과 직접·엑셀 등록 대상을 구분할 수 있다.
3. **정기교육 방식 선택 — 양호**  
   집합교육과 자체교육의 운영 차이를 안내하고 탭 시연이 동작한다.
4. **신청·서명 증빙 — 양호**  
   부서 신청, 대상자 선택, 서명파일, 종료 처리의 선후관계가 연결된다.
5. **채용시교육 — 양호**  
   고용형태별 필요시간과 업무 투입 전 이수 원칙이 별도 흐름으로 보인다.
6. **기타·발생형 교육 — 양호**  
   특별·작업변경·MSDS 등 정기시간에 합산되지 않는 교육을 구분한다.
7. **미달자 마감·독촉 — 양호**  
   미달자 필터 시연, 일괄 독촉, 독촉 이력 확인까지 후속조치가 이어진다.

## 핵심 개선 사항

- 고위 공무원 보고용 `3분 핵심 시연`을 별도 제공한다.
- 핵심 시연은 `위험 현황 → 자동 선별 → 실행·증빙 → 책임 추적` 4장면으로 구성한다.
- 첫 장면에 전체 이수율·미이수자·30일 내 마감·최우선 조치 부서를 실시간 KPI로 표시한다.
- 각 장면에 발표자가 그대로 읽을 수 있는 `보고 포인트`를 제공한다.
- 마지막 독촉 이력에는 환경과·공공시설사업소·건설과의 조치 샘플 3건을 제공한다.
- 기존에는 7개 메뉴의 시작점과 종료점이 드러나지 않아 초보 사용자가 업무 순서를 스스로 조합해야 했다.
- 모든 안전보건교육 화면에 `7단계 업무 튜토리얼` 진입 버튼을 추가했다.
- 튜토리얼은 화면을 실제로 이동하며 `지금 할 일`과 `완료 기준`을 함께 보여준다.
- 집합/자체교육 탭과 미달자 필터는 튜토리얼 안의 시연 버튼으로 직접 확인할 수 있다.
- 별도 모달을 만들지 않고 본문 인라인 패널을 사용해 단일 모달 규칙을 유지했다.
- 버튼, 제목 구조, `aria-live`, 키보드 포커스 표시, 모션 축소 설정을 반영했다.
