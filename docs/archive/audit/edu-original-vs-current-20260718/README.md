# 최초 원본 대비 현재 안전보건교육 비교

- 비교 기준: 최초 커밋 `eab7f6c` (2026-06-23)
- 현재 기준: 2026-07-18 작업 화면
- 결론: **교육 메뉴의 위치와 3탭 핵심 구조는 유지됐지만, 전체 메뉴 구성과 교육 화면의 보조 UX는 변경됐다.**

## 1. 그대로 유지된 부분

- `안전보건관리체계 → 안전보건교육` 진입 위치
- 안전보건교육이 별도 한 개 메뉴라는 원칙
- 내부 3탭: `교육 일정 / 이수 현황 / 과정 관리`
- 교육 일정 카드와 상태 필터
- 교육 등록 → 대상자 선택 → 출석 확인 → 결과 확정 → 개인 이수시간 반영 흐름
- 개인별 이수현황과 부서 필터
- 과정 관리와 반기 점검 진입
- 제안된 `현업 근로자 / 관리감독자` 상세 메뉴는 문서에만 기록했으며 현재 메뉴에는 적용하지 않음

## 2. 변경된 부분

### 전체 플랫폼 메뉴

최초 원본에서는 위험성평가·유해위험요인·의견청취·개선조치 등이 안전보건관리체계의 형제 메뉴였다. 현재는 일부 기능이 별도 GNB로 승격·재분류되고 시설물 안전관리·인력 평가·예산관리 등이 추가됐다.

따라서 **플랫폼 전체 메뉴 구조는 최초 원본과 동일하지 않다.** 다만 안전보건교육 자체는 계속 안전보건관리체계 아래 단일 메뉴로 남아 있다.

### 안전보건교육 화면

- 화면명: `안전보건교육` → `안전보건교육 이수 관리`
- 요구사항 표기: `SFR-004·010` → RFP 정리본 기준 `SFR-008`
- `제안요청서 최소 시연 범위` 안내 추가
- `3분 핵심 시연`과 `시연 초기화` 추가
- 상단 요약: 이수율·예정·완료·담당부서 → 등록교육·결과확정·연결증빙·이수완료
- 교육 카드에 연결 증빙 건수 표시
- 결과 확정 전에 교육일지·출석부 등 결과 증빙을 필수 확인하는 단계 추가
- 기존 모형 첨부 영역을 실제 파일 선택과 공통 파일 제한 안내로 변경
- 법령 확정값처럼 보일 수 있는 문구를 `관리 기준`, `시연 설정값`, `발주처 협의 후 확정`으로 완화
- 날짜·장소 등 시연 데이터 최신화

## 3. 변경 성격

- 구조 변경: 교육 페이지 내부 3탭에는 없음
- 업무 흐름 변경: 결과 확정 전에 증빙 확인 단계가 1단계 추가됨
- 보조 UX 변경: 튜토리얼·초기화·요약지표·범위 안내 추가
- 데이터 변경: 시연 날짜와 증빙 건수 최신화
- 전체 IA 변경: 교육 외 플랫폼 메뉴는 이후 작업으로 재구성됨

팀장 초기 설계의 핵심인 `단일 교육 메뉴 + 3탭 + 교육 생애주기`는 보존됐다. 그러나 픽셀과 화면 구성까지 원본 그대로인 상태는 아니며, 시연성과 증빙 현실성을 높이기 위한 보조 요소가 추가됐다.
