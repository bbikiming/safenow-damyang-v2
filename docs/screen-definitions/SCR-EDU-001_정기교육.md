# SCR-EDU-001 정기교육

> v6 · 기획 관점 · 안전보건교육 모듈 · 현행 구현 기준(7메뉴 재이식)

## 1. 화면 정보

| 항목 | 내용 |
|---|---|
| 화면 ID | SCR-EDU-001 |
| 화면명 | 정기교육 |
| URL 경로 | edu-reg.html (현업근로자) · edu-sup.html (관리감독자) |
| 상위 메뉴 | 안전보건교육 > 현업근로자/관리감독자 > 정기교육 |
| 화면 유형 | 목록 + 등록/수정 모달 |

- 정기교육을 **집합교육**(재난안전과 등록 → 부서 신청·서명 → 교육 종료 처리)과 **자체교육**(부서가 등록·즉시 완료) 두 방식으로 관리하는 목록 화면이다.
- 정기교육은 개인별 채용일(관리감독자는 지정일) 기준 **사이클**로 관리하며, 사이클 내 인정 교육시간이 필요시간에 도달하면 이수로 판정한다.
- **현업근로자용(edu-reg.html)과 관리감독자용(edu-sup.html)은 동일 화면**이며, 대상 모집단·사이클·필요시간만 다르다(§6).

## 4. 화면 구성별 정리

### 4-1. 서브탭
- 구성: [집합교육] / [자체교육(부서)] — 탭별 필터 상태를 각자 유지.

### 4-2. 필터 바
- 구성: 검색어(교육명·강사·장소), 진행 상태(전체/신청 접수 중/완료), 연도, 월. 자체교육 탭에는 부서 필터 추가.
- 우측 액션: [필터 초기화(적용 수)], [집합교육 등록] 또는 [자체교육 등록·진행].

### 4-3. 교육 카드 목록
- 구성: 상태 배지(진행중/완료), 부서 배지(자체), 제목, 메타(일시·교육시간(다회차면 "N일 합계")·강사·장소·신청 부서 수·인원).
- 이벤트:

| 트리거 | 동작 결과 | 이동 / 메시지 |
|---|---|---|
| [부서 신청] (진행중 집합만) | 부서 신청 모달 | "{부서} · N명 신청 완료" |
| [상세] (집합) | 정기교육 상세로 이동 | edu-reg-detail.html?id= |
| [상세] (자체) | 인라인 상세 모달(신청현황·이력) | — |
| [수정] | 등록 모달 재사용(대상자 재선택 불가) | 회차 변경 시 이수기록 시간 자동 갱신 |
| [삭제] | 삭제 확인 모달 | "신청 N건·이수기록 M건 회수" 명시 후 삭제 |

- 상태: 목록이 비면 "등록된 교육이 없습니다" 또는 "조건에 맞는 교육이 없습니다".

### 4-4. 등록/수정 폼
- 구성: 부서(자체만), 교육 일자·시간(회차 탭), 강사, 장소, 내용, 첨부파일, 교육 사진, 교육 대상자(자체 신규만).
- 회차 탭: 회차별 (일자·시작 시각·종료 시각) 입력, 최대 5일. 회차별 시간이 자동 산정되고 전 회차 합계가 교육시간이 된다. 종료가 시작보다 앞서면 "0시간으로 계산" 경고.
- 상태: 수정 시 대상자는 변경하지 않고 "신청 취소 후 재신청"으로 안내한다.

### 4-5. 부서 신청 모달(집합)
- 구성: 부서(조직도), 근로자 선택, 서명파일 업로드(필수).
- 이벤트: 근로자 미선택·서명 미첨부 시 각각 차단(§5).

### 4-6. 시연 투어 바
- 구성: 발표용 시연 실행·초기화 바. 집합교육 등록 → 부서 신청 → 종료 처리 → 이수 반영을 화면을 건너 안내한다.

### 4-7. 온나라 결재 상신(공통)
- 우측 최상단 [온나라 결재 상신] 버튼 → 결재 문서 미리보기 팝업(인력평가와 동일한 미리보기 UI). 이 화면의 기본 서식은 **① 교육 종류별 목록**이며, 팝업 안에서 3종(교육 종류별 목록·개인별 이수 현황·통합)으로 전환된다.
- 대상 연도·부서 조회 조건, 조직도 기반 결재선 편집, 상신·PDF 저장을 포함한 전체 규격은 **SCR-EDU-005 §4-6(안전보건교육 공통)**을 따른다.

## 6. 정책

- **현업 vs 관리감독자:** 두 화면은 운영 방식이 동일하고, 대상 모집단(현업근로자 / 관리감독자), 사이클(6개월 / 12개월), 정기 필요시간(현업 12시간·사무·판매 6시간 / 관리감독자 16시간)만 다르다. (근거: 산업안전보건법 제29조, 안전보건교육규정)
- **집합 vs 자체:** 집합교육은 등록 후 부서 신청·서명·종료 처리 단계를 거치고, 자체교육은 부서가 등록과 동시에 완료 처리한다.
- **교육시간은 회차에서 파생:** 교육시간은 회차별 시각에서 자동 산정하며, 별도 시간 입력란을 두지 않는다. 회차를 바꾸면 이미 쌓인 이수기록 시간도 함께 갱신한다.
- **인정시간 산정:** 정기교육 인정은 현재 사이클 구간의 정기 교육시간 합산으로 판정한다(기타 교육은 합산 제외).
- **회수 경로:** 교육은 수정·삭제, 부서 신청은 신청 취소(상세 화면)를 제공하며, 삭제·취소 시 이수기록을 연쇄 회수한다.
- **온나라 결재 상신:** 결재 문서 수치는 화면과 동일 데이터에서 파생하며, 결재선은 코드에 고정하지 않고 조직도에서 지정한다(위임전결규칙 소관). 전체 규격은 SCR-EDU-005 §4-6.
