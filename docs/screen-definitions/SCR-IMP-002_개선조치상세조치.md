## 4. 화면 구성별 정리

### 4-1. 개선조치 개요
- 구성: 출처 배지, 관리대상, 유해위험요인(위험성평가 출처 시), 담당자, 기한, 연결 평가.
- 상태: 위험성평가 출처는 평가로 이동하는 연결을 제공한다.

### 4-2. 조치 전 내용
- 구성: 대책 내용, 조치 전 사진.
- 상태: 위험성 추정에서 입력된 대책·사진을 표기한다.

### 4-3. 조치 실시·완료
- 구성: 조치 내용 입력, 조치 후 사진, [완료 처리].
- 이벤트:

| 트리거 | 동작 결과 | 이동 / 메시지 |
|---|---|---|
| [완료 처리] | 조치 완료로 전환 | "조치 완료 처리" |
| 위험성평가 출처 완료 | 해당 공정을 재평가 대기로 전환 | "공정 재평가 대기 전환" 안내 |

- 상태: 완료 처리된 개선조치는 조치 결과·조치 후 사진을 보기 모드로 표시한다.

## 6. 정책

- **조치 증빙:** 조치 전/후 사진으로 이행을 증빙한다. (근거: 산업안전보건법 제36조)
- **재평가 연동:** 위험성평가 출처 개선조치의 완료는 해당 공정을 재평가 대기로 전환한다. 재평가에서 허용이 확인되어야 위험성평가 완료 조건이 충족된다.
- **출처 일원화:** 위험성평가 감소대책과 그 밖 출처(점검·의견·경영방침 점검·사고·수동)를 동일한 조치·완료 흐름으로 처리한다.
