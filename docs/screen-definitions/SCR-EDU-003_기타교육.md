## 4. 화면 구성별 정리

### 4-1. 필터 바
- 구성: 검색어(내용·강사·장소), 교육 분류(특별교육/작업내용 변경 시/MSDS/직무교육/자체·기타), 부서, 연도.
- 우측 액션: [기타 교육 등록·진행].

### 4-2. 카드 목록
- 구성: 분류 배지, 부서 배지, 제목, 상태 배지(완료/진행중), 메타(일정·시간·강사·장소·대상자 수). 카드마다 [상세]·[수정]·[삭제].

### 4-3. 등록/수정 모달
- 구성: 교육 분류, 부서, 일자, 시작 시각, 교육 시간(직접 입력), 강사, 장소, 내용, 대상자(신규만), 증빙 첨부.
- 이벤트:

| 트리거 | 동작 결과 | 이동 / 메시지 |
|---|---|---|
| [등록] | 자체교육으로 즉시 완료 처리 | "{분류} 진행 완료 · N명 카운트" |
| [삭제] | 확인 후 이수기록 연쇄 회수 | "기타 교육을 삭제했습니다 · 이수기록 N건 회수" |

- 상태: 수정 시 대상자는 변경하지 않고 "삭제 후 재등록"으로 안내한다.

### 4-4. 온나라 결재 상신(공통)
- 우측 최상단 [온나라 결재 상신] 버튼 → 결재 문서 미리보기 팝업(인력평가와 동일한 미리보기 UI). 이 화면의 기본 서식은 **① 교육 종류별 목록**이며, 팝업 안에서 3종(교육 종류별 목록·개인별 이수 현황·통합)으로 전환된다.
- 대상 연도·부서 조회 조건, 조직도 기반 결재선 편집, 상신·PDF 저장을 포함한 전체 규격은 **SCR-EDU-005 §4-6(안전보건교육 공통)**을 따른다.

## 6. 정책

- **자체교육 형식·즉시 완료:** 기타 교육은 부서가 등록과 동시에 완료 처리한다.
- **정기 합산 제외:** 기타 교육 실적은 참고로 표시하되 법정 정기 필요시간 인정에는 합산하지 않는다.
- **분류 체계:** 특별교육·작업내용 변경 시·MSDS·직무교육·자체/기타 5종으로 분류한다. 관리감독자용은 채용시·작업내용 변경 시·특별교육을 포함한다. (근거: 산업안전보건법 제29조)
- **입력 구조:** 회차 탭 없이 단일 일자·시각·시간(직접 입력)으로 등록한다(정기·채용시의 회차 탭과 구분).
- **회수 경로:** 삭제 시 이수기록을 연쇄 회수한다(신청 취소·이수 취소 별도 경로 없음).
- **온나라 결재 상신:** 결재 문서 수치는 화면과 동일 데이터에서 파생하며, 결재선은 코드에 고정하지 않고 조직도에서 지정한다(위임전결규칙 소관). 전체 규격은 SCR-EDU-005 §4-6.
