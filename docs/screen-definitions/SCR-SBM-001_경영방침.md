## 4. 화면 구성별 정리

### 4-1. 경영방침 현황 — 등록·수정
- 구성: 읽기 화면 ↔ 편집 화면 전환. 편집 항목 — 적용 대상 구분(전체/사업/사업장)·대상, 경영방침 본문, 안전보건 목표(KPI), 점검자 지정, 증빙 첨부. 보조 — 작성 가이드, 근거법령·고려사항 관리, 점검항목 관리.

| 트리거 | 동작 결과 | 이동 / 메시지 |
|---|---|---|
| [수정] / [편집 닫기] | 읽기 ↔ 편집 화면 전환 | — |
| 적용 대상 구분 선택 | '전체'가 아니면 대상 선택 활성화 | — |
| [선택](대상) | 사업/사업장 목록에서 대상 지정 | 선택 안내 |
| [조직도](점검자) | 조직 구성원에서 점검자 지정 | — |
| [작성 가이드] | 작성 원칙·KPI·게시·점검 안내 표시 | 안내 |
| [근거법령]/[고려사항]/[점검항목] 관리 | 해당 항목 편집 | 현재 저장은 안내만(미반영) |
| 증빙 [파일 추가]/[설명 수정]/[삭제] | 첨부 목록 추가·수정·삭제 | 삭제 시 확인 절차 없음(§7) |
| [임시저장] | 작성 중 상태로 안내 | 현재 입력값 저장 안 됨(§6·§7) |
| [등록] | "경영방침이 등록되었습니다 · 관련 담당자에게 알림 발송" 안내 | 현재 저장·발송은 시뮬레이션 |

- 상태: 읽기 / 편집. 경영방침 본문·목표·적용 대상·점검자·증빙은 현재 저장되지 않음(실제 저장 규칙 §6·§7). 읽기 화면의 적용 대상·증빙·점검자 표시가 실제 입력과 연동되지 않음(§6·§7).

### 4-2. 온나라 결재
- 구성: 결재 상태 표시(미상신/결재중/승인완료/반려), 상신·이력·반려 후 처리.

| 트리거 | 동작 결과 | 이동 / 메시지 |
|---|---|---|
| [온나라 결재 상신] | 결재 상태를 '결재중'으로 전환 + 온나라 결재요청 안내(문서명, 결재선: 팀장→과장→부군수) | 상신 선행조건·실연계 §6·§7 |
| 결재 상태 전환(미상신/결재중/승인완료/반려) | 상태 전환 | 현재는 데모용 수동 전환(권한·결재선 무시) — §6·§7 |
| [결재 이력] | 결재 처리 이력 표시 | 현재 고정 예시 |
| [재상신] / [내용 수정] (반려 시) | 재상신 = '결재중' 전환 / 내용 수정 | — |

- 상태: 미상신 / 결재중 / 승인완료('공표 가능') / 반려. 실제 온나라 연동, 승인·반려 수신 후 상태 갱신·되돌림은 미정(§7).

### 4-3. 게시·주지
- 구성: 게시·주지 보드(게시 전/후), 게시 등록 및 전 직원 알림.

| 트리거 | 동작 결과 | 이동 / 메시지 |
|---|---|---|
| [+ 게시 등록] | 위치·전/후 사진·게시일 입력 후 등록 시 전 종사자 알림 | 현재 알림은 시뮬레이션 |

- 상태: 게시 항목(전/후) 목록 표시. 승인 완료 후에만 게시·공표 허용할지, 알림 대상 인원 산정 기준은 §6·§7.

### 4-4. 반기 이행점검
- 구성: 반기 점검표(항목별 O/X/해당없음 판정), 확정·상신, 부적정 항목의 개선조치 자동 연계.

| 트리거 | 동작 결과 | 이동 / 메시지 |
|---|---|---|
| [이행점검 작성] | 반기 점검표 작성 화면 표시 | — |
| 점검표 [등록] | 작성 중 → 등록(확정 전까지 수정 가능) | 등록 안내 |
| [확정·온나라 결재 상신] | 점검 확정 + 부적정(X) 항목을 개선조치로 자동 등록 + 온나라 결재요청 안내 | "X 항목 N건 개선조치 자동 등록" 안내 |
| [개정] (확정 후) | 새 버전 작성 | — |

- 상태: 작성중 / 등록완료 / 확정. 확정 후에는 읽기 전용. 전 항목 판정 강제 여부, X 판정 시 사유 필수 여부, 개선조치 자동 등록 시 기한·담당자 산정 규칙은 §6·§7.

## 6. 정책

- **상신 선행조건** — 미작성·미등록 상태에서도 상신이 가능. 등록 완료 등 상신 가능 선행 상태를 둘지 확정 필요(§7).
- **공표·게시 선행조건** — 결재 승인 완료와 무관하게 게시·알림이 가능. 승인 완료 후에만 공표·게시를 허용할지 확정 필요(§7).
- **개선조치 자동 연계** — 이행점검에서 X(부적정) 항목을 개선조치로 자동 등록할 때 기한·담당자 산정 규칙 확정 필요(§7).
- **점검항목 기준 단일화** — 경영방침 현황에 표시되는 점검항목 수와 실제 반기 점검표의 항목 수가 다름. 점검항목 마스터 기준 단일화 필요(§7).
- **표시값 산출 기준** — 경영방침 현황의 적용 대상·증빙·점검자 표시가 실제 입력과 연동되지 않고 고정 표시됨. 동적 산출 여부 확정 필요(§7).
- **대상 인원 산정** — 등록 완료 알림 대상 수, 공표 대상(전 종사자) 인원이 고정 수치로 표시됨. 실제 수신자·대상 인원 산정 기준 확정 필요(§7).
- **온나라 문서번호 채번** — 문서번호 채번 규칙·단일 출처 확정 필요(§7).
- **공표 기능 노출** — 게시·주지와 별도의 '전 직원 공표' 기능을 화면에 둘지 확정 필요(§7).
