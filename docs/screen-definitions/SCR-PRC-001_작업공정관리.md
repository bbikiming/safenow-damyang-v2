# SCR-PRC-001 작업공정 관리

> v6 · 기획 관점 · 위험성평가 모듈(작업공정)

## 1. 화면 정보

| 항목 | 내용 |
|---|---|
| 화면 ID | SCR-PRC-001 |
| 화면명 | 작업공정 관리 |
| URL 경로 | rsk-proc.html |
| 상위 메뉴 | 위험성평가 > 작업공정 관리 |
| 화면 유형 | 목록 |

- 관리대상(시설·사업장)의 **작업 공정**을 등록·관리하고, 각 공정에 **설비·유해인자·유해위험요인**을 매핑하는 화면이다. 여기서 정리된 공정과 유해위험요인이 위험성평가의 **원본**이 된다.
- 공정은 카드로 나열되며, 카드마다 하나의 [수정]으로 공정명·담당 평가자·설명과 설비·유해인자·유해위험요인을 **카드 안에서** 편집한다.
- 유해위험요인은 **공정·설비·유해인자 세 축이 각각** 표준 유해위험요인을 끌어와 **합집합**으로 자동 구성되며, 사용자가 가감하거나 직접 추가할 수 있다.
- **확정(잠금) 단계가 없다.** 저장하면 즉시 활성이며, 정기 위험성평가는 저장된 현재 공정 전체를 대상으로 읽는다. 설비·작업방법이 바뀌면 반영 시 수시 위험성평가 생성을 확인한다.

## 4. 화면 구성별 정리

### 4-1. 관리대상 선택
- 구성: 관리대상 선택기, 소속 업종·공정 건수 요약, [공정 추가].
- 이벤트:

| 트리거 | 동작 결과 | 이동 / 메시지 |
|---|---|---|
| 관리대상 변경 | 그 대상의 공정 카드만 표시 | 편집 중이면 취소하고 목록 갱신 |
| [공정 추가] | 목록 상단에 새 편집 카드 표시 | — |

### 4-2. 공정 카드(요약)
- 구성: 순번·공정명·표준/직접 구분, 담당 평가자, 설비·유해인자·유해위험요인 요약 칩, [수정]·[삭제].
- 상태: 담당 평가자 미지정 시 경고 표기. 요약은 조밀하게 1~2줄로 노출.

### 4-3. 공정 편집(카드 내부)
- 구성: 공정명·담당 평가자·설명 입력, 설비/유해인자/유해위험요인 영역, [취소]·[반영].
- 이벤트:

| 트리거 | 동작 결과 | 이동 / 메시지 |
|---|---|---|
| [설비 선택] / [유해인자 선택] | 카드 안에서 검색·다중선택 패널을 펼친다 | 적층 팝업을 띄우지 않는다 |
| 선택 적용 | 선택 항목 반영 + 유해위험요인 자동 합산 | "유해인자/설비 N개 반영" |
| [직접 추가] | 유해위험요인 직접 입력 패널을 펼친다 | 출처(직접/점검) 선택 |
| 요인 체크 해제 | 해당 요인을 제외 대상으로 표시 | 반영 시 제외 |
| [반영] | 변경 저장 | 델타에 따라 수시평가 확인(§4-5) |
| [취소] | 편집 취소·원복 | — |

### 4-4. 유해위험요인 자동 구성(병렬 합집합)
- 구성: 공정·설비·유해인자 각각이 끌어온 표준 유해위험요인의 합집합. 각 요인에 출처(표준/직접/점검)와 법령근거 상태 표기.
- 상태: 법령근거가 확정되지 않은 항목은 "법령 매핑 대기"로 표기한다.

### 4-5. 반영 시 수시 위험성평가 확인
- 구성: 변경 요약·대상 공정·변경 사유를 담은 확인 대화.
- 이벤트:

| 트리거 | 동작 결과 | 이동 / 메시지 |
|---|---|---|
| 설비 변경·작업방법 변경·점검 발견 요인 추가로 반영 | 수시 위험성평가 생성 확인을 요청 | "이 변경으로 수시 위험성평가를 생성합니다" |
| 확인 | 열린 수시 평가에 변경 공정 추가, 없으면 신규 생성 | 위험성평가 목록에서 확인 안내 |
| 유해위험요인 정리(가감)만 반영 | 저장만 수행 | 수시평가 사유 아님 안내 |

## 6. 정책

- **확정 개념 폐기:** 별도 확정·개정 단계를 두지 않는다. 저장 즉시 활성이며, 정기 위험성평가는 저장된 현재 공정 전체를 대상으로 읽는다.
- **자동매핑 원칙:** 유해위험요인은 공정·설비·유해인자 세 축이 **각각 독립적으로** 표준 유해위험요인을 끌어온 합집합이다. 설비에서 유해인자를 자동으로 파생시키지 않으며, 설비·유해인자는 각각 선택한다.
- **수시평가 트리거(법정 사유 한정):** 설비 추가·변경·제거, 작업방법·절차 변경, 점검에서 발견된 유해위험요인 추가만 수시 위험성평가 사유로 본다. 유해위험요인 목록을 정리(가감)하는 것 자체는 사유가 아니다. (근거: 산업안전보건법 제36조, 사업장 위험성평가에 관한 지침 제15조)
- **법령근거:** 표준 자료에 법령이 포함되지 않아 자동매핑·직접·점검 유입 항목은 기본 "법령 매핑 대기"로 두고, 법령 매핑에서 후속 확정한다.
