## 1. 화면 정보

| 항목 | 내용 |

| 화면명 | 채용시교육 |
| URL 경로 | edu-hire.html |
| 상위 메뉴 | 안전보건교육 > 현업근로자 > 채용시교육 |

- 현업근로자의 채용시교육 이수/미이수를 **근로자 단위**로 관리하고, 미이수자를 선택해 일괄 이수 처리하거나 별도 교육을 추가 등록하는 화면이다.
- 근로자를 명단에 등록하면 채용시교육은 별도 등록 없이 자동으로 **미이수** 상태로 파생되며, 이수 처리 시 고용형태·계약기간별 필요시간을 인정한다.
- 채용일 이후에 이수하면 **지연 이수**로 구분해 표시한다.

## 4. 화면 구성별 정리

### 4-1. 서브탭
- 구성: [미이수(건수)] / [이수(건수)] / [교육 건(건수)] — 탭 전환 시 선택·이수 구분 필터 초기화.
- 미이수·이수는 **근로자 단위**, 교육 건은 **실시한 채용시 교육 단위** 뷰다(§4-7).

### 4-2. 필터 바
- 구성: 검색어(이름·부서), 부서, 고용형태, 채용연도. 이수 탭에는 이수 구분(정상 이수/지연 이수) 추가.
- 우측 액션: [채용시 교육 추가], (미이수 탭) [선택 이수처리(선택 수)].

### 4-3. 묶어보기 바
- 구성: 부서별 / 채용일별 / 안 묶기. 그룹 헤더에 인원수와 그룹 전체선택.

### 4-4. 근로자 표
- 구성: (미이수 탭 체크) 이름 · 부서 · 고용형태(계약개월 병기) · 채용일 · 필요시간 · 이수일 · 상태 · (이수 탭) [이수 취소].
- 상태 어휘: 정상 이수(이수일이 채용일 이전), 지연 이수(이수일이 채용일 이후), 미이수.
- 이벤트:

| 트리거 | 동작 결과 | 이동 / 메시지 |
|---|---|---|
| 헤더 체크 | 필터 결과 전체선택/해제 | — |
| [선택 이수처리] | 일괄 이수처리 모달 | 선택 0명이면 "미이수자를 1명 이상 선택하세요." |
| [이수 취소] (이수 탭) | 확인 후 미이수로 되돌림 | "N건 이수 취소 · 미이수로 되돌렸습니다." |

### 4-5. 일괄 이수처리 모달
- 구성: 선택 인원 명단, 이수일 방식(각자 채용일 처리(권장) / 일괄 이수일 지정), 일괄 이수일·시작 시각(방식 2), **교육 건 생성 방식**, 강사·주관, 내용.
- 상태: 일괄 이수일이 채용일보다 늦으면 지연 이수로 처리한다.
- **교육 건 생성 방식:** [근로자별 개별 건](기본 · 1명당 교육 건 1개) / [한 건으로 묶기].
  - 묶기는 이수일이 하나여야 성립하므로 선택 시 **일괄 이수일 지정으로 자동 전환**되고, 각자 채용일로 되돌리면 묶기는 자동 해제된다.
  - 묶더라도 **필요시간(1h·4h·8h)이 다른 인원은 법정 교육시간이 다른 별개 과정**이라 건이 나뉜다. 모달에 예상 생성 건수와 시간별 인원을 미리 보여준다.
  - 묶인 건은 특정 부서에 속하지 않으므로 부서는 비우고 **부서별 신청 행**으로 나눠 기록하며, 이수기록은 근로자별로 각자 필요시간을 인정한다.

### 4-6. 채용시 교육 추가 모달
- 구성: 회차 탭(행사 진행 시간), 강사·주관, 장소, 내용, 교육 대상자(부서별 미이수자 선택), 첨부·사진.
- 상태: 인정 시간은 대상자 고용형태별 필요시간으로 부여한다(회차 합계와 별개).

### 4-7. 교육 건 탭 (채용시 교육 목록)
- 구성: 필터(교육명·강사 검색 + 부서 + 연도), [채용시 교육 추가], 안내 문구, 교육 건 카드 목록.
- 카드: 부서 칩 · 교육명 · 진행 상태(완료/진행중) · **온나라 결재 상태 칩**(§4-8) / 일시 · 시간 · 강사 · 대상자(인원·이름).
- 목적: 채용시교육은 근로자 단위 화면이라 교육 건 목록이 없어 **교육별 결재 상신**을 걸 자리가 없었다. 이 탭에서 건 단위로 보고 건별로 상신한다.
- 대상자 이수 취소는 이 탭이 아니라 **이수 탭**에서 처리한다(회수 경로 단일화).
- 교육 건은 **일괄 이수처리 시 근로자당 1건**, **[채용시 교육 추가] 시 다수 대상 1건**으로 생성된다.

### 4-8. 온나라 결재 상신(공통)
- 우측 최상단 [온나라 결재 상신] 버튼 → 결재 문서 미리보기 팝업(인력평가와 동일한 미리보기 UI). 이 화면의 기본 서식은 **① 교육 종류별 목록**이며, 팝업 안에서 3종(교육 종류별 목록·개인별 이수 현황·통합)으로 전환된다.
- 대상 연도·부서 조회 조건, 조직도 기반 결재선 편집, 상신·PDF 저장을 포함한 전체 규격은 **SCR-EDU-005 §4-6(안전보건교육 공통)**을 따른다.
- 이 화면은 **교육 건 탭(§4-7)의 카드**에 교육별 상신 결재 상태 칩을 노출한다(미상신=[결재 상신], 이후 상태 칩→상태 팝업). 교육 1건의 실시 결과 문서.

## 6. 정책

- **필요시간 산정:** 일용은 1시간, 기간제 계약 1주 이하는 1시간, 1주 초과~1개월은 4시간, 그 밖은 8시간을 채용시교육 필요시간으로 인정한다. (근거: 산업안전보건법 제29조, 안전보건교육규정 별표)
- **업무 투입 전 이수 원칙:** 채용시교육은 업무 투입 전 이수를 원칙으로 하며, 채용일 이후 이수는 지연으로 표시한다.
- **미이수 자동 파생:** 근로자를 명단에 등록하면 채용시교육 미이수가 자동 파생되어 이 화면과 이수현황에 즉시 나타난다.
- **회수 경로:** 이수 취소는 **그 근로자의 이수기록만** 회수해 미이수로 되돌린다. 여러 명이 묶인 교육 건은 삭제하지 않고 유지하며(다른 대상자의 이수가 사라지면 안 되므로), 대상자가 모두 빠져 빈 건이 된 경우에만 교육 건을 함께 회수한다.
- **온나라 결재 상신:** 결재 문서 수치는 화면과 동일 데이터에서 파생하며, 결재선은 코드에 고정하지 않고 조직도에서 지정한다(위임전결규칙 소관). 전체 규격은 SCR-EDU-005 §4-6.
