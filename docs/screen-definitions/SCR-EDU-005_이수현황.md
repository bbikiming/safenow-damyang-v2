## 4. 화면 구성별 정리

### 4-1. 뷰 전환(서브탭)
- 구성: [부서별 요약] ↔ [대상자별 상세] (기본 요약).

### 4-2. 부서별 요약 뷰
- 구성: 필터(부서명 검색 + 구분·고용형태·채용연도 + 완료율 구간(전체/100%/70% 이상/70% 미만)), [독촉 이력].
- 표: 부서 · 대상 인원 · 완료 인원 · 완료율 게이지(100%·70% 이상·70% 미만 색 구분) · [상세].

### 4-3. 대상자별 상세 뷰
- 구성: 필터(검색(이름·부서·구분) + 부서 + 구분·고용형태·채용연도 + 출처 + 이수 상태(전체/미이수(미달)/기한초과/마감 임박/완료)), [일괄 독촉(선택 수)], [독촉 이력].
- 표: (미이수 체크) 이름 · 부서 · 구분 · 고용형태 · 채용일 · 현재 사이클(시작~종료) · 종료까지 남은 기간 배지 · 필요 · 인정 · 미달 · 정기 상태 · **개인 결재**(§4-6 개인별 상신 칩 — 미상신이면 [개인 상신], 이후 온나라 결재 상태 칩).
- 정렬: 미달자 우선, 그다음 마감 임박순.
- 상태 어휘: 남은 기간 배지(완료 / 기한초과 / 마감 임박(30일 이내) / 여유), 정기 상태(완료 / 미이수).

### 4-4. 모집단 집계 안내
- 구성: 적용 필터에 따른 "집계 대상 · N명 중 완료 M명" 문구.
- 상태: 구분·고용형태·채용연도·출처는 완료율 재집계의 모집단 기준으로 함께 적용된다. 검색·완료율 구간은 표시 행만 좁히고 합계는 모집단 전체를 유지한다.

### 4-5. 독촉
- 구성: 미이수자 선택 → [일괄 독촉] → 부서별 명단·추가 메모 → 발송. [독촉 이력]로 일시·부서·인원·대상자·메모 열람.
- 이벤트:

| 트리거 | 동작 결과 | 이동 / 메시지 |
|---|---|---|
| [일괄 독촉] | 부서별로 묶어 독촉 발송·이력 기록 | "N개 부서, M명 독촉 발송 완료" |
| 미이수자 선택 0명 | 발송 차단 | "미이수자를 1명 이상 선택하세요." |

### 4-6. 온나라 결재 상신 — 결재 문서 미리보기(안전보건교육 공통)

> 이 절이 안전보건교육 전 화면(SCR-EDU-001~006)의 온나라 결재 상신 공통 기준이다. 각 화면은 진입 지점과 대상 범위만 다르고 문서 미리보기 팝업의 구조(결재선·상신·인쇄)는 동일하다.

- **목적:** 시스템이 보유한 교육·이수 데이터로 규격 기안문을 자동 생성해, 담당자가 손으로 옮겨 적지 않고 대상·결재선만 정해 상신하도록 한다. 문서 양식 확정(컨펌)을 위한 화면이다.
- **상신 트리거 3종:**

| 트리거 | 진입 위치 | 대상 | 문서 |
|---|---|---|---|
| ① 총괄 상신 | 목록 화면 우측 최상단 [총괄 결재 상신] 버튼 | 대상 연도·부서 범위의 모든 총괄 데이터 | 한 팝업 3탭 — 교육 종류별 목록 / 개인별 이수 현황 / 통합 (아래 표) |
| ② 교육별 상신 | 각 교육 카드·상세의 결재 상태 칩, 상세 상단 [이 교육 결재 상신] | 교육 1건(그 N월 집합/자체/기타/관리감독자 교육) | 그 교육의 실시 결과 보고(개요·회차·참석/이수·서명·증빙) |
| ③ 개인별 상신 | 이수현황 대상자별 상세 표의 각 인원 행 결재 칩 | 대상자 1명 | 그 사람의 교육 이수 확인서(이수 내역·인정시간·판정) |

- **총괄(①) 서식 3종(팝업 안 탭 전환):**

| 서식 | 문서명 | 본문 구성 |
|---|---|---|
| 교육 종류별 목록 | {연도}년 안전보건교육 실시 결과 보고 | 종합 요약 → 교육 종류별 실시 집계(합계 행) → 종류별 상세 목록(일시·교육명·시간·강사·장소·대상 부서·이수·상태) |
| 개인별 이수 현황 | {연도}년 안전보건교육 개인별 이수 현황 보고 | 종합 요약 → 부서별 이수 현황 → 개인별 현황(사이클·필요·인정·미달·이수 여부) → 미이수자 조치 계획 |
| 통합 | {연도}년 안전보건교육 실시 결과 및 이수 현황 통합 보고 | 위 두 서식을 한 건의 기안문으로 묶음 |

- **총괄 조회 조건(팝업 상단):** 대상 연도 · 대상 부서(전체 또는 1개). 바꾸면 문서가 즉시 다시 생성되고, 바뀐 조건이 문서 부제·개요표에 그대로 표기된다.
- **화면별 총괄 기본 서식:** 정기·채용시·기타·정기교육 상세 = 교육 종류별 목록, 이수현황·근로자 명단 관리 = 개인별 이수 현황. 어느 화면에서든 탭으로 전환 가능.
- **온나라 결재 상태(목록 노출):** 상태 4단계 **미상신 → 결재중 → 결재완료 / 반려**. 교육 카드·상세(교육별 상태)와 이수현황 대상자 행(개인별 상태)에 칩으로 노출한다. 미상신이면 칩 자리가 [결재 상신]·[개인 상신] 액션 버튼이고, 상신 이후엔 상태 칩이며 클릭 시 결재 상태 팝업(문서번호·상신일·결재선·상태 이력 + 시연 회신 [결재 완료 처리]·[반려 처리] + [문서 다시 보기]·[재상신])이 열린다. 실제 연계 시 완료·반려는 온나라가 회신하며, 시연 버튼은 프로토타입용이다.
- **결재선(팝업 안, 조회 조건 아래):** 법령은 결재권자를 정하지 않고 **지자체 위임전결규칙 소관**이므로 고정하지 않는다. 기본값은 기안자 소속 부서의 **팀장(검토) → 부서장(결재)** 이며 조직도에서 직위명으로 파생한다. 1단계 기안은 현재 로그인 사용자로 고정, 2단계부터 [조직도](공용 인라인 조직도)에서 지정한다. [단계 추가](최종 결재자 앞에 검토 삽입, 기안 외 최대 3단계)·[삭제]·[기본값]. 단계 이름은 마지막이 결재, 그 앞은 검토로 자동 부여. 결재선을 바꾸면 기안문 결재란·개요표가 즉시 갱신되고, 지정값은 세션 동안 다른 교육 화면으로 이동해도 유지된다.

## 6. 정책

- **모집단 일관 집계:** 구분·고용형태·채용연도·출처 필터는 표시 대상과 완료율 재집계에 동일하게 적용해, 필터를 건 화면의 수치가 전체 수치로 오독되지 않게 한다.
- **사이클 기반 판정:** 대상자별 필요시간과 현재 사이클 인정시간을 비교해 미달·완료를 판정하고, 남은 기간으로 마감 임박·기한초과를 표시한다.
- **독촉 대상:** 독촉은 미이수자만 선택 가능하며, 발송 결과는 이력으로 보존한다.
- **대표 진입:** 이수현황이 안전보건교육 그룹의 대표 진입 화면이며, 구 교육 주소는 이 화면으로 유도한다.
- **결재 문서 = 화면 수치:** 결재 문서(§4-6)의 모든 수치는 화면과 동일한 데이터에서 파생하며 결재용 별도 값을 만들지 않는다. 결재선은 코드에 고정하지 않고 조직도에서 지정한다(위임전결규칙 소관).
