## 4. 화면 구성별 정리

### 4-1. 교육 정보 헤더
- 구성: 제목·상태(진행중·신청 접수 / 완료), 메타(구분·일정·교육시간(다회차 "N일 합계")·강사·장소·신청 부서·인원), 다회차 회차별 일정, 첨부·사진, (진행중) [부서 신청]·[교육 종료 처리].

### 4-2. 신청현황 표
- 구성: 부서 · 신청자 명단 · 인원 · 서명파일 · 신청일 · [신청 취소](행별).
- 상태: 신청이 없으면 "아직 신청이 없습니다".
- 이벤트:

| 트리거 | 동작 결과 | 이동 / 메시지 |
|---|---|---|
| [부서 신청] | 부서·근로자·서명 입력 모달 | "{부서} · N명 신청 완료" |
| [교육 종료 처리] | 확인 후 신청자 전원 이수 반영·완료 전환 | "교육 종료 · N명 카운트 완료" |
| [신청 취소] | 확인 후 신청·이수기록 회수 | 완료 교육이면 "반영된 이수시간도 회수" 경고 |

### 4-3. 이력
- 구성: 일시·유형·메모·발신 타임라인.

### 4-4. 온나라 결재 상신(공통)
- 우측 최상단 [온나라 결재 상신] 버튼 → 결재 문서 미리보기 팝업(인력평가와 동일한 미리보기 UI). 이 화면의 기본 서식은 **① 교육 종류별 목록**이며, 팝업 안에서 3종(교육 종류별 목록·개인별 이수 현황·통합)으로 전환된다.
- 대상 연도·부서 조회 조건, 조직도 기반 결재선 편집, 상신·PDF 저장을 포함한 전체 규격은 **SCR-EDU-005 §4-6(안전보건교육 공통)**을 따른다.

## 6. 정책

- **집합교육 공용 상세:** 현업근로자·관리감독자 집합교육이 같은 상세 화면을 쓰며, 교육 종류로 대상 모집단을 구분한다.
- **종료 처리 비가역:** 교육 종료 처리는 신청자 전원에게 교육시간을 이수 반영하며, 되돌릴 수 없음을 확인 모달에 명시한다.
- **신청 취소 회수:** 신청 취소 시 신청과 이수기록을 함께 회수하고, 완료 교육이면 반영된 이수시간도 회수한다.
- **온나라 결재 상신:** 결재 문서 수치는 화면과 동일 데이터에서 파생하며, 결재선은 코드에 고정하지 않고 조직도에서 지정한다(위임전결규칙 소관). 전체 규격은 SCR-EDU-005 §4-6.
