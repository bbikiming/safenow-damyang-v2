# 기획 — 안전보건교육 재설계: 결정 필요사항 v1

2026-07-16. 설계 착수 전 확정이 필요한 항목 정리.
법정 기준은 산업안전보건법 시행규칙 §26 및 [별표4](2023-09-27 개정 반영) 기준으로 조사함.
⚠ 표시는 발주처(담양군)·노무사 확인이 필요한 항목.

---

## A. 법적 검토 필요 항목

### A-1. 정기교육 주기 — "채용일 기준"이 아니라 "반기 기준"일 가능성 높음 ⚠
2023-09-27 시행규칙 개정으로 근로자 정기교육은 **매분기 → 매반기**로 변경됨.
- 사무직 종사 근로자: **매반기 6시간 이상**
- 판매업무 직접 종사 근로자: **매반기 6시간 이상**
- 그 외 근로자(현업업무종사자 대부분 해당): **매반기 12시간 이상**
- 관리감독자: **연간 16시간 이상** (반기 아님)

→ 인터뷰의 "채용일 기준 계산"은 법령상 근거를 못 찾음. 정기교육은 역법상 반기(1~6월 / 7~12월) 단위 카운트가 표준 실무.
**결정 필요**: ① 반기 단위 카운트로 확정할지, ② 채용 반기(입사한 반기)의 정기교육 의무 처리 방식 — 채용 시 교육(8h) 이수 시 해당 반기 정기교육을 면제/대체로 볼지는 명시 규정을 확인하지 못함. **고용노동부 질의 또는 노무사 확인 필요.**
- 시스템 기본값(제안): 반기 단위 카운트 + 채용 반기는 채용 시 교육만 필수, 정기교육은 다음 반기부터 발생. 확인 후 규칙만 교체 가능하게 계산식을 분리 구현.

### A-2. 채용 시 교육 시간 — 고용형태별 차등
- 일용근로자·계약 1주 이하 기간제: **1시간 이상**
- 계약 1주 초과 ~ 1개월 이하 기간제: **4시간 이상**
- 그 밖의 근로자: **8시간 이상**
- 시점: **업무 투입(배치) 전** 실시.

**결정 필요** ⚠: 담양군 계약직의 계약기간 분포(기간제 다수?) — 근로자 등록 시 '고용형태·계약기간' 입력을 받아 필요시간을 자동 산정할지.

### A-3. 현업업무종사자 범위 ⚠
공무원은 산안법 일부 적용 제외지만, 지자체 소속 **현업업무종사자**(청소·환경, 시설관리, 도로보수, 상하수도, 조리 등)는 안전보건교육 의무 적용 대상.
**결정 필요**: 담양군이 지정한 현업업무종사자 직종·부서 목록 확보 (재난안전과 보유 여부 확인). 시스템의 '현업근로자' 명단 범위가 이것으로 정해짐.

### A-4. 관리감독자 하위 메뉴 구분 (분석·제안)
별표4 기준 관리감독자 교육과정은 아래 4종. 이걸 하위 메뉴로 제안:
| 과정 | 법정 시간 | 비고 |
|---|---|---|
| 정기교육 | 연간 16시간 이상 | 총 시간의 1/2 이상 집체·현장·비대면 실시간 |
| 채용 시 교육 | 8시간 이상 | 관리감독자 지정 시 |
| 작업내용 변경 시 | 2시간 이상 | |
| 특별교육 | 16시간 이상 | 별표5 유해위험작업 해당 시 |

**제안 메뉴**: 관리감독자 → **정기교육 / 기타 교육**(채용·작업변경·특별을 기타로 통합, 분류 셀렉트) — 현업근로자와 구조 통일.
**결정 필요** ⚠: ① 이 구분으로 갈지, 4종 전부 노출할지. ② 담양군 관리감독자 지정 범위(팀장급? 사업소장?) 명단 확보.

### A-5. 기타 교육 분류(셀렉트) 후보 — 법정 과정 기준
특별교육(별표5 대상작업) / 작업내용 변경 시 교육(2h) / 물질안전보건자료(MSDS) 교육 / 직무교육(안전보건관리책임자 등) / 비법정·자체 기타.
**결정 필요**: 이 5개 분류로 확정할지, 담양군 실무 분류 추가 여부.

### A-6. 교육시간 인정 기준 ⚠
- 무재해 사업장 감면: 전년 무재해 시 다음 해 정기교육 50% 감면 가능 — 적용할지.
- 온라인/비대면 교육 인정 범위(관리감독자 정기는 1/2 이상 집체 등 제한).
**결정 필요**: 인정시간 계산에 감면·방법 제한을 반영할지, 단순 합산으로 갈지 (프로토타입 기본값: 단순 합산 + 감면 미적용).

## B. 시스템 설계 결정 항목 (법령 무관)

### B-1. 메뉴 구조 (제안)
```
안전보건교육
├ 현업근로자
│  ├ 정기교육 (집합교육 · 자체교육)
│  ├ 채용시교육
│  └ 기타 교육
├ 관리감독자
│  ├ 정기교육
│  └ 기타 교육 (A-4 확정에 따름)
├ 이수현황 (부서/대상자별 필요·인정·미달·완료여부 + 독촉)
└ 근로자 명단 관리 (인사연동 + 부서별 직접 등록·엑셀 업로드)
```
**결정 필요**: 이수현황·명단 관리를 현업/관리감독자 공통 메뉴로 둘지(제안), 각 하위에 둘지.

### B-2. 근로자 명단 데이터
- 인사정보시스템 연동분(공무원·공무직): 읽기 전용, `DYV2.ORG` 파생.
- 계약직: 부서 담당자가 직접 등록 또는 **엑셀 업로드**. 재난안전과는 등록 불가(파악 불가 전제) — 조회만.
- 필수 속성: 이름, 부서, 구분(사무직/비사무직/판매/관리감독자), 고용형태(정규·공무직·기간제·일용), 채용일, 계약기간(기간제), 재직상태.
**결정 필요**: 퇴직자 처리(이력 보존·카운트 제외 시점), 부서 이동 시 이수시간 승계 여부.

### B-3. 집합교육 신청 흐름
재난안전과 등록(일정·강사·장소·내용) → 부서가 대상자 선택 신청 + **서명파일 업로드** → 교육 종료 처리 시 신청 대상자에 시간 카운트.
**결정 필요**: ① 정원·마감일 개념 필요한지. ② 서명파일은 신청 시 1회인지, 교육 후 참석확인 서명인지(시점). ③ 불참자 처리(신청했으나 미참석 → 카운트 제외) 필요한지.

### B-4. 독촉 기능
이수현황에서 미이수자 개별 선택 또는 일괄 독촉 → 알림 목업 + 이력 기록.
**결정 필요**: 위험성평가 재촉처럼 **부서 담당자의 사유 회신**까지 받을지, 단순 알림 발송만 할지.
