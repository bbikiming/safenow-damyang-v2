## 1. 핵심 설계 결정 — 전자문서 공통 엔진 (e-Doc)

> 원칙: **전자문서로 분류된 59건은 전부 프로그램 화면에서 값을 입력**한다. 파일 업로드로 갈음하지 않는다.

### 1.1 표준 폼 7종 (59건 전자문서를 유형으로 흡수)

엑셀 분류본의 59건은 전부 "등록·조회·수정" CRUD에 계획(P)/결과(C)/조치(A) 패턴이다.
문서마다 화면을 만들지 않고 **표준 폼 7종**으로 흡수하며, 필드 가감은 [시스템 관리 > 프리셋 양식 관리]에서 한다.

| # | 폼 유형 | 주요 필드 | 흡수 문서 (예) |
|---|---|---|---|
| F1 | **계획서** (P) | 대상연도·기간 / 목표 / 세부 추진내용 / 일정 / 담당 / 첨부 | 교육 계획 16건, 의견청취 계획, 전담조직 활동 계획 등 |
| F2 | **실시결과** (C) | 실시일 / 대상·참여 / 결과 요약 / 정량 실적 / 증빙 첨부 | 각종 "실적·결과" 문서 |
| F3 | **회의록** | 일시·장소 / 참석자(조직도 선택) / 안건 / 의결사항 | 산업안전보건위원회, 협의체 (분기·월간) |
| F4 | **교육결과** (특화) | 과정유형 / 일시·장소·강사 / 대상자 명단 / 출석 체크 / 교안 첨부 | 교육 결과 21건 |
| F5 | **점검표** | 항목별 Y/N·O/X / 미흡 사유 / 조치 입력 | 수립·이행 점검, 의무이행 점검 |
| F6 | **평가서** | 평가 항목·배점 / 점수 / 판정 / 의견 | 책임자 수행평가, 수급인 평가 |
| F7 | **조치보고** (A) | 발생 원인(연계 표시) / 조치 내용 / 기한·담당 / 확인자 / 증빙 | 교육 점검 후 조치, 후속 조치(포상·인사) |

### 1.2 문서 상태 머신 (전 전자문서 공통)

```
작성중(임시저장) → 등록완료 → 확정 (확정 시 온나라 결재 요청 팝업 1회)
```
- 모든 전이는 이력(누가·언제·무엇을)에 기록 — COM-010 감사로그.
- **확정 문서는 수정 불가**, [개정] 버튼으로 새 버전 생성 (버전 이력).

### 1.3 회차(일정) 자동 관리

- 문서 마스터의 주기(연간/반기/분기/월간/수시/발생시)로 **연도별 회차를 자동 생성** → 대시보드 시기도래·달력에 노출.
- 회차 미등록 + 기한 임박 → COM-009 자동 알림 (D-30/D-7/D-1/초과).
- "발생시" 문서는 회차 없이 [+ 등록] 버튼으로 건별 생성.

### 1.4 ~~서명 체계~~ → 온나라 결재로 갈음 (컨펌 결과)

**컨펌 결정**: 제안요청서가 교육자 서명을 요구하지 않으므로 서명 체계는 구현하지 않는다.
교육 결과 등 모든 전자문서는 **확정 시 온나라 결재 요청**으로 갈음하며, "온나라로 결재 요청을 보냈습니다" 안내 팝업이 1회 표시된다.
출석은 담당자가 명단에서 체크하고, 출석자의 이수 시간은 결과 확정 시 자동 가산된다.

## 3. 메뉴별 버튼 플로우 (9개 대메뉴)

### 3.1 경영방침 [SFR-005]
```
[목표·방침 작성](F1) → [임시저장]/[등록] → [온나라 결재 상신] → 결재완료(상태 연동 표시)
→ [공표](게시 + 전 직원 알림) → 반기 [이행점검 작성](F5 O/X) → X 항목 → [개선조치 생성](U2)
```

### 3.2 조직 [SFR-006·009·010]
```
조직도(행정포털 연계 조회) → [선임 등록](안전·보건관리자 폼 + 선임계 첨부) → 미선임 경고 뱃지
[전담조직 활동 계획](F1)/[활동 실적](F2) — 발생시
반기: [수행평가 작성](F6, 평가자=경영책임자 지정) → 판정 미흡 → [후속조치 등록](F7: 포상/인사) → 개선조치 집계
```

### 3.3 위험성평가 [SFR-007] — 기존 UX 유지 (이미 이식 완료)
연계 보강만: 평가 확정 → 결과 문서가 이행문서함에 자동 등록(REVIEW→C), X항목 개선조치 자동 생성(기존).

### 3.4 유해·위험요인 관리 [SFR-007·019] — 기존 proc-list 유지 + 신고 보강

[+ 신고 접수](QR·온라인 폼) → 접수 → [담당 분배] → 검토 → U2 분기(수시평가 생성/개선조치 생성/반려)
→ 상태 스테퍼(U1) + 연동 정보 카드(U3) / 작업환경측정 결과: [측정 결과 등록](F2 + 측정기관 보고서 첨부)
```

### 3.5 안전보건교육 [SFR-004·010] — ★ 전용 화면 승격 (상세 4장)

[+ 의견 등록](근로자: QR·온라인 / 담당자: 대리 입력) → 접수대기

```
[+ 사업 등록] → e호조 계약 조회(연계 61건) 불러오기 → [적격 수급인 평가](F6, 계약 전)

## 7. 질의 결과 (2026-06-11 컨펌)

① 서명 방식 → **서명 제외** (제안요청서 미요구). 교육 결과 등은 확정 시 온나라 결재 요청 — 안내 팝업 1회.

③ 온나라 결재 범위 → **확정 문서 전반**.
④ 기획 전체 → **기획대로 진행** (구현 완료).
