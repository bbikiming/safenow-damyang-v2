`담양군_프로토타입_v2_재구축_프롬프트.md` (2026-06-11) 기준 재구축 + v2.1 재기획 반영.
원본(`프로토타입 - 하늘팀장님/`)은 수정하지 않았으며, 디자인 토큰·레이아웃 엔진과

## 실행

정적 HTML — 폴더에서 바로 열거나 `python3 -m http.server` 후 `index.html` 접속.

## v2.1 재기획 원칙 (2026-06-11 피드백 반영)

1. **위험성평가는 기존 프로토타입 UX 그대로** — 등록된 유해·위험요인(구 "공정") 기반으로
   평가방법을 고르고 평가를 진행하며, 부적정(X) 항목에 개선조치를 입력하는 플로우.
   `proc-list → risk-list → risk-create(평가방법 선택) → risk-regular(회차 진행) → risk-process-eval(요인별 작성·개선조치 입력) → risk-detail`
2. **PDCA는 문서 구분축이지 업무 화면이 아니다** — 대메뉴 화면에서는 PDCA를 노출하지 않고,
   **업무문서 > 이행문서(docs-preset.html) 한 곳에서만** 단계 필터 칩 + 세트 묶어 보기로 제공.
3. **지자체 사용성: 진입점·뎁스 최소화** — 대메뉴 클릭 즉시 "할 일(업무 기능)"이 보인다.
   탭 2개(업무 기능 / 문서) 이상 만들지 않음. 메뉴 깊이 GNB > 사이드바 2뎁스.
